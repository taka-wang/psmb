# psmb/tcp

Proactive service for modbus tcp library.

## Install

```
go get -u github.com/taka-wang/psmb/tcp
```


## Environment variables

- CONF_PSMBTCP: config file location
- EP_BACKEND: remote service discovery endpoint (optional)
