# mgo-history

Mongodb-based history data store

## Install

```
    go get -u github.com/taka-wang/psmb/mgo-history
```

## Environment variables

- CONF_PSMBTCP: config file location
- EP_BACKEND: remote service discovery endpoint (optional)
